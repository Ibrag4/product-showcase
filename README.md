# Pokedex — Product Showcase (Desafio)

Pequena Pokedex construída com React + TypeScript + Vite. Consome a [PokeAPI](https://pokeapi.co/) para listar e mostrar detalhes dos Pokémons.

## Tech
- React + TypeScript (Vite)
- Fetch API (PokeAPI)
- CSS moderno (arquivo `App.css`)

## Funcionalidades
- Lista de Pokémons (grid responsivo)
- Busca por nome (filtro em tempo real)
- Paginação simples
- Modal com detalhes (imagem, altura, peso e tipos)
- Loading com animação de Pokébola

## Como rodar localmente
```bash
# instalar dependências
npm install

# rodar em dev (http://localhost:5173)
npm run dev
